# Tremor parser

Funcion python para parsear archivo csv proveniente del MARN para poblar base de datos de Tremor

## Variables de entorno

Las variables de entorno requeridas son:

- DB_USER
- DB_PASSWORD
- DB_HOST
- DB_PORT
- DB_NAME
